The Document Object Model

  • Basis of HTML5 is "New features should be based on HTML, CSS, the DOM and Javascript."
  • The DOM provides common tree like structures that all pages should follow.
  • Computer Scientists love trees (the mathematical kind) because you can test them.
  • The Document Object Model (DOM) is a programming API for HTML and XML documents. It defines the logical structure of documents and the way a document is accessed and manipulated. In the DOM specification, the term "document" is used in the broad sense - increasingly, XML is being used as a way of representing many different kinds of information that may be stored in diverse systems, and much of this would traditionally be seen as data rather than as documents. Nevertheless, XML presents this data as documents, and the DOM may be used to manage this data.




Block line & Inline line level elements

Block-level elements begin on a new line, stacking one on top of the other & occupy any available width, Block-level elements may be nested inside one another & wrap inline-level elements. We'll most commonly see block-level elements used for larger pieces of content, such as headings or paragraphs.

Heading & the paragraph tags are given as <h1>...</h1> to <h6>...</h6> and <p>...</p>

Inline-level elements do not begin on a new line. They fall into normal flow of a document, lining up one after the other, and only maintaining the width of their content. Inline-level elements may be nested inside one another; however, they cannot wrap block-level elements. We'll usually see inline-level elements with smaller pieces of content, such as few words.

About HTML building structure

HTML5 introduced new structurally based elements including the <header> <nav> <article> <section> <aside> <footer> elements. All of these new elements are intended to give meaning to the organization of our pages & improve our structural semantics. They are all block-level elements & do not have implied position or style.

Creating Hyperlinks
Hyperlinks provides the ability to link from one webpage or resource to another. Hyperlinks are established using the anchor <a> inline elements.
Example: <a href = "https:facebook.com">Facebook</a>
Adding Multimedia
Multimedia includes images, videos, and audio files. These can be added using <img> and <video> tags. Most common attributes used in these tags include height, width, autoplay, loop & controls.
Example for images: <img src = "imgs/redcross-logo.png" alt = "Red Cross Logo"/> — for local reference.
<img src = "http://www.redcross.org/images/logo.png" alt = "Red Cross Logo"/> — for Absolute reference.
Creating tables
Tables are used in HTML for organizing the data in a more convincing fashion so as to better understand it. It is created using <table> <thead> <tfooter> <tbody> <tr> and <td>tags.
