Introduction

In this course, you are going to learn about the absolute basics of HTML5 or if we elaborate it: HyperText Markup Language and this is going to be your introduction to not only HTML but, also to the World of Web Design. The instructor has designed this course very well, it is basically divided into three segments or three weeks of study I should say.

  1. Week 1: How the internet works!
    • In the week one of the course, you learn about the evolution of HTML, about how we went from HTML1 to HTML5 over the course of the years. The browsers, and about the browser wars. We will also see the difference between Syntax & Semantics in HTML5. Then, you learn about what happens when you enter an address into the url. Something called as — The Request/Response Cycle. It's interesting to learn about what happens behind when you want access a webpage.
  2. Week 2: Let's learn how to code in HTML5!
    • This is where we learn how to code. It's the most important segment of the course. It's where you get your hands dirty. You'll be introduced to enormous number of HTML tags. You'll be learning about how to add headings, paragaphs, lists, links, tables, multimedia such as images, videos and it's controls onto your web page.
  3. Week 3: Accessibility; Validating your code & hosting it on the Internet!
    • By this time, we'll be atleast knowing how to create a basic HTML file which is syntactically & symantically correct. Now what? We'll learn about why accessability & validating your code is very important. Then, we'll move on to the most exciting part — Hosting your page on the web. Using the free/paid hosting services and domain names, we can launch your first HTML page on the internet for the world to witness!
