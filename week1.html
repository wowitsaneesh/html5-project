WEEK 1: HOW THE INTERNET WORKS!

This week will uncover the "mystery" behind the Internet. What happens when you type a URL into your browser so that a webpage magically appears? What is HTML5 & what happened to HTML 1 - 4?

HTML stands for "HyperText Markup Language." It is way of marking up files so that browsers such as Internet Explorer, Safari, Google Chrome, Firefox et cetera know how to display your content as a webpage. HTML uses tags to distinguish between content (what the user should see) & the instructions to displaying them. A webpage isn't just one file but many pieces put together by your browser using something called the request response cycle.

The difference between Syntax & Symantics.

Let's see the difference between the syntax & Symantics — Syntax is the arrangement of elements and attributes to create well-formed documents. Now, coming to Symantics — Semantics is concerned with meaning. In HTML, this is the purpose of elements and attributes, and the logical (sense and reference) relationship between elements and the attributes of those elements.

The Request/Response Cycle: What happens when you type into a URL.

The web is a cycle of requests and responses that flow between clients and servers.

This is one of the most fascinating concept while you start to learn how the internet works. To know & understand this cycle, you have to know what a client & the server is. You also have to know what a HTTP Request is. HTTP stands for HyperText Transfer Protocol. There's also a thing called HTTPS. Now, what's that? It's a fascinating concept.
